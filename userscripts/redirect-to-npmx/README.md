## Description

[npmx.dev](https://npmx.dev) is "a fast, modern browser for the npm registry". It is a better alternative to [npmjs.com](https://www.npmjs.com/).

When you visit any `npmjs.com` page, this userscript tries to redirect you to npmx.dev. That's all it does.

## Installation

To use this userscript, you'll need a userscript manager like [Tampermonkey](https://www.tampermonkey.net/) (available for Chrome, Firefox, and other browsers).

1. Install Tampermonkey or a similar userscript manager for your browser.
2. Go to [the GreasyFork Page](https://greasyfork.org/en/scripts/585069), and click on the "Install this script" button.
3. Visit any `npmjs.com` page, and you should be redirected to the corresponding `npmx.dev` page.

## Alternatives

There is a [similar script](https://github.com/wolfcreative/npmx-link-router) that does a bit more: it lives on every page you visit (not just `npmjs.com`) and preprocesses links before you even click on them. Personally I find it a bit overkill.
